# Тренировачный макет Cootels
## Описание
  макет с использованием парадигмы "ООП" для укрепления и закрепления материала основных 
  концепций данной парадигмы. Код JS разделен на несколько категорий путей. Основным файлом js 
  является index.js с типом "module", где и подключены классы, лежащие в папке components.

  Практически весь код написан на ванильном JS, за исключением слайдера с использованием
  библиотеки Swiper. 

  Используется основная методология БЭМ, разработанная компанией "Yandex"

##  Технический стек

- HTML, основной язык разметки, гиперссылок, различных семантических тегов
- CSS, каскадная таблица стилей для стилизования сайта
- БЭМ, "БЛОК | Элемент | Модификатор", методология, разработанная компанией "Yandex"
  [https://ru.bem.info/methodology/]
- JavaScript, мультипарадигменный язык программирования, для создания сайтов, веб-приложений.
- JSON, "JavaScript Object Notation", текстовый формат обмена данными, основанный на JavaScript
  [https://www.json.org/json-en.html]
- "ООП", методология программирования
- Swiper, библиотека для создания слайдеров
